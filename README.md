# Rocket News

Criar um formulário de matrícula responsivo pra uma escola fictícia chamada Trybewarts.

## Instalação

```bash
# Clonar o projeto
$ git clone https://github.com/pedroygor/RocketNews.git

# Entre no diretório
$ cd RocketNews

# Abra o arquivo index.html no navegador
```

## Stack utilizada

HTML, CSS e javascript
